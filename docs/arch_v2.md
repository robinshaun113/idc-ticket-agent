# P2 架构图 · Excalidraw 绘制指南

> 打开 https://excalidraw.com 绘制，完成后导出 PNG 到 `docs/arch_v2.png`

## 画布布局（1200 × 800）

### 左区（0–200）：入口
- **矩形** `工单输入` — 浅蓝填充，标注"客户反馈 / 监控告警 / 巡检发现"
- 右箭头指向 Classifier

### 中左区（250–450）：分类层
- **菱形** `Classifier` — 橙色填充，标注"类别 × 优先级 × 置信度"
- 三条出边，分别标注 P0 / P1-P3 / confidence<0.6

### 中区（500–600）：诊断层
- **矩形** `Diagnoser` — 绿色填充，标注"调用工具 × 3"
- 三个小矩形挂在 Diagnoser 下方（排成一行）：
  - `get_monitoring`（监控数据）
  - `get_cmdb`（资产配置）
  - `get_history_tickets`（历史工单）
- 用虚线连接三个工具到 Diagnoser

### 中右区（650–800）：知识层
- **矩形** `KB-Searcher` — 紫色填充，标注"复用 P1 RAG\n检索 SOP + 历史方案"
- 右箭头指向 Output

### 右上区（650–750，上面）：升级分支
- **圆角矩形** `Escalator` — 红色填充，标注"P0 工单\n立即升级通知"

### 右区（850–950）：输出层
- **矩形** `Output` — 灰色填充，标注"处理方案\n• 根因分析\n• SOP 引用\n• 操作建议"

### 底部（Day 29 虚线框）：
- 用虚线框标注 `Reviewer 回环（Day 29 加入）`，包含：
  - **菱形** `Reviewer` — 标注"置信度 < 0.6？"
  - 回路箭头回到 Diagnoser，标注"retry with hints"

## 配色建议
- 分类层：`#FFA500`（橙）
- 诊断层：`#4CAF50`（绿）
- 知识层：`#9C27B0`（紫）
- 升级：`#F44336`（红）
- 输出：`#607D8B`（灰蓝）
- 回环（虚线）：`#999`
